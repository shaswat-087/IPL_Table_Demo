# IPL_Table_Demo


A Flask web app that scrapes live IPL 2026 standings from Wikipedia
and displays them in a styled, color-coded table.

## Features
- Live data scraped from Wikipedia using BeautifulSoup
- Team-specific brand colors (CSK yellow, MI blue, RCB red, etc.)
- Auto-sorted by points and NRR
- Clean, responsive table UI

## Tech Stack
- Python, Flask, Jinja2
- BeautifulSoup, Cloudscraper
- HTML/CSS

## Run Locally
pip install -r requirements.txt
python app.py

Then open http://127.0.0.1:5000
